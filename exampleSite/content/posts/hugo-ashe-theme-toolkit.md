`hugo-ashe` 的目标不是“再一个主题”，而是“可落地的网站主题工具箱”。

## 设计目标

- 快速上线：最小配置即可运行
- 内容优先：阅读体验优先于装饰
- 渐进增强：默认轻量，可按需启用评论/统计

## 你会得到什么

- 首页文章流 + 分页
- 文章详情页与相关推荐
- JSON 搜索索引 + 前端搜索弹窗
- 图片灯箱（PhotoSwipe）
- 可选评论系统（Giscus）

## 推荐工作流

1. 在站点仓库中以子模块方式引入主题。
2. 用 `exampleSite` 验证主题更新是否破坏基础功能。
3. 通过 GitHub Actions 自动构建并发布展示站点。

如果你希望把主题二次封装到团队模板中，建议基于此仓库创建上游分支，并保留原始变更历史。
