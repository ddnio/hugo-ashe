# Contributing

感谢你为 `hugo-ashe` 做贡献。

## 本地开发

```bash
git clone https://github.com/nio/hugo-ashe.git
cd hugo-ashe
hugo server --source exampleSite --themesDir ../.. --theme hugo-ashe
```

## 提交前检查

```bash
hugo --source exampleSite --themesDir ../.. --theme hugo-ashe --minify
```

## 提交建议

- 使用清晰的提交信息（例如：`feat: add xxx`、`fix: resolve xxx`）
- 每个 PR 尽量聚焦单一主题
- 涉及视觉变更时，请附截图或说明
